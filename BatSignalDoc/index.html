Overview

This is the documentation for Project Bat Signal ("Bat Signal").

Bat Signal allows individuals to seek help by filling out a Google Form. A post is then automatically generated and delivered to the #bat-signal slack channel.

Details

Bat Signal has the following deliverables:

Slack single post
Creates a Slack post each time someone raises the bat signal through the Google Form.

Technical Details

Bat Signal is a Google Apps Script project, written in the Google Apps Script (GAS) language, which is essentially javascript.

A "From spreadsheet - On form submit" trigger is used to automatically create the Slack posts.

Understanding the files

Helper.js
Contains dictionaries of variables which are used throughout the project.

Slack.js
Script for Slack posts. The "onFormSubmit" function, which creates the posts, is triggered by a "From spreadsheet - On form submit" trigger.
